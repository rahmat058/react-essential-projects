## Shared Stack

- **Node.js 24.11.0**
- **React 19** + **TypeScript**
- **Tailwind CSS 4** — glassmorphism theme
- **Redux Toolkit** — predictable state
- **Framer Motion** — smooth animations
- **lucide-react** — icons
- **Mock API** — swap-ready for real backend

## Study Path

1. Build or run the project locally
2. Read `ARCHITECTURE.md` — understand data flow
3. Practice explaining debounce, keyboard nav, and API cancellation
4. Review `INTERVIEW-QUESTIONS.md` before mock interviews
